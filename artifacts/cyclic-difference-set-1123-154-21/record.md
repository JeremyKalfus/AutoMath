## statement_lock
Determine whether the cyclic group C_1123 admits a (1123,154,21)-difference set.

## definitions
- Work in the additive cyclic group `G = Z/1123Z`. A `(1123,154,21)` difference set is a subset `D ⊂ G` of size `154` such that every nonzero `g ∈ G` has exactly `21` representations `g = d_1 - d_2` with `d_i ∈ D`.
- The order parameter is `n = k - λ = 154 - 21 = 133 = 7 * 19`.
- A numerical multiplier lane is the natural one here: if `t` is a `1123`-multiplier in the Baumert-Gordon sense, then a putative difference set satisfies `tD = D + s` for some `s ∈ G`.
- Because `1123` is prime, any normalized multiplier action `tD' = D'` with `t ≠ 1 mod 1123` forces `D'` to be a union of multiplication-by-`t` orbits on `G`; the nonzero orbits all have size `ord_1123(t)`, while `{0}` is the only fixed orbit.
- Since `154 = 14 * 11` and `153 = 9 * 17`, an orbit obstruction will be decisive as soon as we can force a normalized multiplier of order `r` with `r ∤ 154` and `r ∤ 153`.

## approach_A
- Structural / invariant lane: use Baumert-Gordon Theorem 3.2 on the full quotient `w = 1123`.
- Since `n = 7 * 19`, it is enough to find a residue `t mod 1123` that is simultaneously a power of `7` and a power of `19`. Then `t` is a `1123`-multiplier.
- If additionally `t ≠ 1 mod 1123`, then after translating by a suitable `a` solving `(t-1)a ≡ -s (mod 1123)`, we may assume `tD' = D'`.
- On the prime group, that would force `|D'| = 154` to equal either `m r` or `1 + m r`, where `r = ord_1123(t)`.
- So the solve reduces to: find such a common-power multiplier `t` and show its order `r` is incompatible with `154` and `153`.

Outcome of the first pass: the common-power intersection only gives the order-`11` subgroup generated by `7`, so that sublane alone does not contradict `154 = 14 * 11`.

Stronger refinement used next: the local repo already treats the standard prime-multiplier theorem as available input. Since `19` is prime, `19 | n = 133`, and `gcd(19,1123)=1`, `19` itself is a numerical multiplier. This is strictly stronger than the common-power lane and is enough on the prime group.

## approach_B

- Then the difference-set equation becomes an exact orbit-incidence problem on the nonzero `t`-orbits together with the optional zero orbit.
- The extremal hope is that even if `r` divides `154` or `153`, the induced class-count equations would still fail because the nonzero difference multiplicities would be too coarse. This is a secondary lane only; it is heavier than the direct orbit-size obstruction and should not be the first computation.

3. For such a multiplier, `tD = D + s` for some `s`.
4. Since `1123` is prime, `t ≠ 1 mod 1123` implies `gcd(t-1,1123)=1`, so translation normalizes to a set `D'` with `tD' = D'`.

7. If `r ∤ 154` and `r ∤ 153`, contradiction. Hence no such difference set exists.

## result
Main theorem candidate from this solve pass:

`The cyclic group C_1123 does not admit a (1123,154,21)-difference set.`

Proof attempt:

1. Assume `D ⊂ Z/1123Z` is a cyclic `(1123,154,21)` difference set.
2. Here `n = k - λ = 133 = 7 * 19`.
3. The repo's standard multiplier lane applies: since `19` is prime, `19 | n`, and `gcd(19,1123)=1`, the prime-multiplier theorem makes `19` a numerical multiplier. Hence
   `19D = D + s`
   for some `s ∈ Z/1123Z`.
4. Because `1123` is prime and `19 ≠ 1 (mod 1123)`, `18` is invertible modulo `1123`. Choose `a` with
   `(19 - 1)a ≡ -s (mod 1123)`.
   Then for `D' = D + a`,
   `19D' = 19D + 19a = (D + s) + 19a = D + a = D'`.
   So after translation we may assume the putative difference set is `19`-invariant.
5. The multiplicative order of `19` modulo `1123` is `561`. Indeed `561 = 3 * 11 * 17`, and the bounded arithmetic check gave
   - `19^33 ≡ 26 ≠ 1 (mod 1123)`,
   - `19^51 ≡ 343 ≠ 1 (mod 1123)`,
   - `19^187 ≡ 33 ≠ 1 (mod 1123)`,
   - `19^561 ≡ 1 (mod 1123)`.
6. Therefore multiplication by `19` on `Z/1123Z` has exactly one fixed point, namely `0`, and every nonzero orbit has size `561`.
7. Any `19`-invariant subset of `Z/1123Z` must therefore have cardinality either `m * 561` or `1 + m * 561`.
8. But `154` is neither a multiple of `561` nor congruent to `1 mod 561`. Contradiction.

Hence no cyclic `(1123,154,21)` difference set exists.

Self-check after the proof attempt: the contradiction is clean and paper-shaped provided the prime-multiplier theorem invocation is verified against the exact Baumert-Gordon / classical statement during verify.
