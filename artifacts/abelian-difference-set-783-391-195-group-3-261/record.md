## statement_lock
Determine whether the abelian group C_3 x C_261 admits a (783,391,195)-difference set.

## definitions
Write `G = P x H` with `P = C_3 x C_9` of order `27` and `H = C_29` of order `29`.

A `(783,391,195)` difference set in `G` is a subset `D ⊂ G` of size `391` such that every nonidentity element of `G` occurs exactly `195` times as an ordered difference `d_1 d_2^{-1}` with `d_i in D`.

For every nonprincipal character `chi` of `G`, the standard character relation gives
`|chi(D)|^2 = k - lambda = 196`, hence `|chi(D)| = 14`.

Because `gcd(2,783)=gcd(7,783)=1` and `2,7 | 196`, both `2` and `7` are numerical multipliers. Since `gcd(2-1,783)=1`, any hypothetical difference set is translate-equivalent to one satisfying `D^(2)=D`. I work with such a normalized translate.

Self-check:
- The multiplier normalization changes only the translate, not existence.
- All later quotient counts are therefore existence-obstructions, not artifacts of a special coordinate choice.

## approach_A
Structural / invariant route.

1. Normalize by the `2`-multiplier so `D^(2)=D`.
2. Contract `D` to quotients where the remaining character field is small enough that `2`-invariance forces character values to be rational.
3. In the quotients of orders `27` and `29`, the nonprincipal Fourier coefficients then become `pm 14`, which collapses the possible coset counts to two-point distributions.
4. Use those distributions together with the `7`-multiplier to reduce the search to a tiny orbit family under the multiplier group.
5. If that family is empty, we get a clean obstruction theorem; if nonempty, the remaining candidates are explicit enough to test directly.

Key deductions already obtained:
- In the quotient `G/H ≅ P`, every nonprincipal character value is fixed by the full Galois group of `Q(zeta_9)`, hence is an integer of absolute value `14`.
- Therefore the `29`-coset counts are forced to be exactly one coset of size `1` and twenty-six cosets of size `15`.
- In the quotient `G/P ≅ H`, every nonprincipal character value is fixed by the full Galois group of `Q(zeta_29)`, hence is also `pm 14`.
- Therefore the `27`-coset counts are forced to be exactly one coset of size `27` and twenty-eight cosets of size `13`.

Self-check:
- The inversion calculation is rigid: the quotient coefficients must be integers, and the congruence leaves only two possibilities in each quotient.
- These conclusions do not yet prove nonexistence, but they already isolate a very narrow theorem-facing slice.

## approach_B
Construction / contradiction route.

Assume a normalized `D` exists and translate so that the exceptional `29`-coset and the full `27`-coset meet at `(0,0)`.

Then the incidence matrix of `D` in `P x H` has:
- one special `P`-row of size `1`,
- twenty-six other `P`-rows of size `15`,
- one special `H`-column of size `27`,
- twenty-eight other `H`-columns of size `13`.

Equivalently,
- `D` contains the full column `P x {0}`,
- the special row contributes only `(0,0)`,
- on the remaining `26 x 28` submatrix every row has sum `14` and every column has sum `13`.

This already forces all characters that are nontrivial on exactly one factor to take the exact values `-14` or `+14`, so only genuinely mixed characters remain to be checked. That is the right contradiction zone: any failure must now come from how the mixed part couples the `3`-primary and `29`-primary coordinates.

Self-check:
- The matrix reformulation is equivalent to the quotient constraints.
- It preserves the micro-paper objective because a contradiction here would separate `[3,261]` from the known `[3,3,87]` case by exact group structure rather than by parameter numerology.

## lemma_graph
Lemma skeleton.

1. Multiplier normalization lemma:
   A hypothetical difference set is translate-equivalent to one with `D^(2)=D`.

2. Quotient-to-`P` lemma:
   If `S` is the contraction of `D` to `P`, then every nonprincipal character of `P` takes value `pm 14` on `S`, so the `29`-coset counts are `1` once and `15` otherwise.

3. Quotient-to-`H` lemma:
   If `T` is the contraction of `D` to `H`, then every nonprincipal character of `H` takes value `pm 14` on `T`, so the `27`-coset counts are `27` once and `13` otherwise.

4. Secondary multiplier lemma:
   Since `7` is also a multiplier and `D^(2)=D`, any relation `D^(7)=Dg` forces `D=Dg`; because `391` is divisible by none of `3,9,29,87,261`, necessarily `g=1`, hence `D^(7)=D`.

5. Orbit-reduction lemma:
   Under the multiplier group generated by `2` and `7`, each nonzero row subset is a union of exactly two size-`7` orbits in `H^*`, and the whole candidate family reduces to choosing `2` of `4` orbit-blocks on each of `4` order-`3` row-pairs and `3` order-`9` row-sextets.

6. Final obstruction target:
   Check whether any such orbit choice satisfies the full difference-set equations for mixed characters.

Self-check:
- Lemmas 1 to 4 are proved on paper up to straightforward bookkeeping.
- Lemma 5 is the first place where a bounded computation is useful rather than optional.
